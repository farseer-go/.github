## 什么是farseer-go
针对 `golang` 平台下的一套技术框架。

我们为您选型出目前最为流行的组件，并按模块化来提供使用这些组件。

框架完美支持 `DDD领域驱动` 的战术设计，如`仓储资源库`、`应用层事务`、`领域事件`、`应用层动态WebAPI`。

> 每个组件都是单独的包，因此版本号也是单独发布的
